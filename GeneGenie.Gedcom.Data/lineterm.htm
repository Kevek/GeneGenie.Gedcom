Heiner Eichmann's GEDCOM 5.5 Sample Page: Different Line Terminators

The GEDCOM standard explicitly allows four different line terminators: CR, CR+LF, LF and LF+CR. The following four transmissions contain in essence the data given in SIMPLE.GED but with all four possibilities to terminate a line.

Line terminator: carriage return
Line terminator: carriage return + line feed
Line terminator: line feed
Line terminator: line feed + carriage return
